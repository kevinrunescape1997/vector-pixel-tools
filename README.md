# SVG Pixel-Rect Optimizer (GUI)

Cross-platform GUI tool to optimize “pixel art stored as SVG `<rect>` elements” by merging rectangles and stripping excess metadata.

## Downloads
Go to the **Releases** page and download the build for your platform:
- Windows: `svg_pixel_rect_optimizer_gui-Windows.zip`
- macOS: `svg_pixel_rect_optimizer_gui-macOS.zip`
- Linux: `svg_pixel_rect_optimizer_gui-Linux-x86_64.AppImage`

## Windows
1. Download and unzip.
2. Run `svg_pixel_rect_optimizer_gui.exe`.

> Windows may show SmartScreen warnings because the app is not code-signed.

## macOS
1. Download and unzip.
2. Drag `svg_pixel_rect_optimizer_gui.app` into Applications (optional).
3. Open it.

> macOS may block unsigned apps. If blocked:
> - System Settings → Privacy & Security → “Open Anyway”.

## Linux (AppImage)
1. Download `svg_pixel_rect_optimizer_gui-Linux-x86_64.AppImage`
2. Make it executable:
   ```bash
   chmod +x svg_pixel_rect_optimizer_gui-Linux-x86_64.AppImage
